# Sudoku-in-Terminal-using-Backtracking

It's a game of Sudoku, runs on the terminal or command line. The main attraction is the usage of Backtracking. Also, one weird problem I had with inheritance, where any changes I made to it's parent classes within that class was not "in session", and did not update on the go?

## How to play
1. Download the files and get them onto your IDE
2. Run the Sudoku.java file, it contains the main function
3. Follow the instructions, type "help" if you don't know what to do
4. When asked to input position, input should be "12", "45", "82", etc, just look at the board
5. Then you are asked to enter a number, enter any from 1-9, whichever u think is right

The Board file consists of the correct board
The Solver randomises that and prints it onto the terminal
The FileSaver is there to just save data and all, I could just do it without creating another class, but it's a college project. The only thing it does right now is print the board onto a file.
